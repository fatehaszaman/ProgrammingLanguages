# Programming Languages

Coursework implementations from Loyola University Chicago's Programming Languages course — exploring non-mainstream paradigms including functional and logic programming.

## Contents

- **`Haskell/`** — Functional programming exercises: factorial, Fibonacci, binary search, array manipulation
- **`Prolog/`** — Logic programming exercises
- **`src/`** — Additional source files
- **`lexicon.txt`** — Language reference material
- **`program *.txt`** — Sample program outputs and traces
